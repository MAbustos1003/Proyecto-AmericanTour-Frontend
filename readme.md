# American Tour S.A.S - Plataforma Web

## Descripción del Proyecto
Este proyecto es la interfaz web (Frontend) para la empresa de transportes American Tour S.A.S., ubicada en Cajicá. Permite a los usuarios navegar por los servicios, registrarse, iniciar sesión y gestionar sus reservas.

## Ambiente de Desarrollo (Herramientas utilizadas)
* **Editor de Código (IDE):** Visual Studio Code.
* **Control de Versiones:** Git y GitHub.
* **Lenguajes:** * HTML5 

## Estructura del Proyecto
El proyecto está dividido en módulos reutilizables sigue buenas prácticas de organización:
* `/css`: Archivos de hojas de estilo.
* `/imgagenes`: Recursos gráficos.
* `index.html`: Página de inicio.
